# 关键词

| 关键词          | 说明              |
| ------------ | --------------- |
| method       | 指明action方法名称    |
| condition    | 指明condition方法名称 |
| next         | 下一步操作脚本         |
| if           | judge关键词        |
| check        | judge关键词        |
| loop         | loop关键词         |
| while        | loop关键词         |
| return\_flag | 返回值标记关键词        |
| store        | 存储器关键词          |
