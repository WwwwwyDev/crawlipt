description: 任何标识符均会按照一定规则进行替换
---

# 标识符

| 标识符                             | 说明              |
| ------------------------------- | --------------- |
| \_\_PRE\_RETURN\_\_             | 接受最近action方法返回值 |
| \_\_rf-{your return \_flag}\_\_ | 接受return\_flag  |
| \_\_v-(your variable}\_\_       | 接受variable      |
| \_\_not-(your condition}\_\_    | 将condition结果取反  |
