# 标识符

| 标识符                             | 说明                |
| ------------------------------- | ----------------- |
| \_\_PRE\_RETURN\_\_             | 最近action方法返回值接受变量 |
| \_\_rf-{your return \_flag}\_\_ | return\_flag接受变量  |
| \_\_v-(your variable}\_\_       | variable接受变量      |
